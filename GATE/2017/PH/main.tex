\item Let $\vec{X}$ be a column vector of dimension $n>1$ with at least one non-zero entry. The number of non-zero eigenvalues of the matrix $\vec{M}=\vec{X}\vec{X}^{\top}$ is
\hfill {(PH 2017)}
\begin{enumerate}
\begin{multicols}{4}
\item 0
\item $n$
\item 1
\item $n-1$
\end{multicols}
\end{enumerate}
