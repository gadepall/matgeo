\item If $\vec{A}=\myvec{ a & b \\ c & -a }$ is a matrix such that $\vec{A}^2=\vec{I}$, where $\vec{I}$ is an identity matrix, then which of the following is TRUE?
\hfill (PI 2024)
\begin{enumerate}
	\begin{multicols}{4}
    \item $1 + a^2 + bc= 0$
    \item $1 - a^2 + bc= 0$
    \item $1 - a^2 - bc= 0$
    \item $1 + a^2 - bc= 0$
    \end{multicols}
\end{enumerate}
