\item Consider a two-port network with the transmission matrix $\vec{T}=\myvec{A & B\\ C & D}$. If the network is reciprocal, then
 \hfill(EC 2016)
\begin{multicols}{4}
\begin{enumerate}
    \item $\vec{T}^{-1}=\vec{T}$
    \item $\vec{T}^2 = \vec{T}$
    \item \(\det(\vec{T})=0\)
    \item \(\det(\vec{T})=1\)
\end{enumerate}
\end{multicols}
